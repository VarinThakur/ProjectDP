Steps to run the project -

1. Activate the virtual environment

    On Windows
    `\env\Scripts\activate`

2. Go to the ProjectDP directory

    `cd ProjectDP`

3. Run the App (/plotter endpoint for finding the plot)

    `python manage.py runserver`
